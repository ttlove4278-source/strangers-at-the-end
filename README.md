## 关于

一部关于世纪末、哲学与选择的群像RPG。

1999年夏，日本虚构都市「櫂町」。一种名为"哲学症"的文明病正在蔓延。
患者被思想附身，获得超常力量，最终化为理论结晶。

你是夏目珀。已死327次。每次都记得。
加缪说人必须想象西西弗是幸福的。
但没人问过西西弗自己怎么想。
